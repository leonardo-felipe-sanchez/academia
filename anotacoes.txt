Quem é o FastAPi?
Framework FastAPI, alta performance, fácil de aprender,fácil de codar, pronto para produção. FastAPI é um moderno e rápido (alta performance)
framework web para construção de APIs com Python 3.6 ou superior, baseado nos type hints padrões do Python.

Async
Código assíncrono apenas significa que a linguagem tem um jeito de dizer para o computador / programa que em certo ponto, ele terá que esperar
por algo para finalizar em outro lugar
